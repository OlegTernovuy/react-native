## Get started

1. Install dependencies

    ```bash
    npm install
    ```

2. Start the app

    ```bash
     npx expo start
    ```

3. Start the app on Android

    ```bash
     npm run android
    ```

4. Start the app on the IOS simulator

    ```bash
     npm run ios
    ```

5. Start the app in a web browser

    ```bash
     npm run web
    ```
